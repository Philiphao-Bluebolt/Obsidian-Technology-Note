图是一种网状的数据结构，由边（Edge）连接顶点（Vertex）组成，常用于描述数据点之间的关联性。与数据、树不同，之间并没有明确的顺序或层次结构。图论是主要研究图的数学分支，数学上，图由包含一组顶点集$V$和一组边集$E$的集合描述：
$$G=\{V,E\}$$
值得一提的是，树本质上是带层次结构的无向无环图。另外，有一种两个顶点之间可以存在多条边的特殊图，称为多重图或伪图，这里不考虑。

## 基本概念 Basic Concepts

+ **顶点和边 Vertex & Edge**
	+ **相关 Related** - 边与它连接的两个顶点相关
	+ **相邻 Adjacent** - 相邻顶点之间直接有边相连
	+ **度 Degree** - 顶点连接的边总数
		+ **入度 Indegree** - 指向顶点的边总数
		+ **出度 Outdegree** - 顶点指向外的边总数



+ **有向性 Directiveness**
	+ **有向图 Directed** - 所有边都是单向边的图
	+ **无向图 Indirected** - 所有边都是双向边的图
	+ **基图 Base** - 有向图对应的无向图称为其基图
	+ **混合图 Hybrid** - 既有有向边又有无向边的图

+ **有环性 Cyclicality**
	+ **有环图 Cylic** - 存在回路路径的图
	+ **无环图 Acylic** - 不存在回路路径的图

+ **连通性 Connectivity**
	+ **连通图 Connected** - 任意两个顶点之间存在路径，即对于任意顶点$x,y$，$x$都能到达$y$
	+ **非连通图 Unconnected** - 存在顶点$x,y$，$x$不能到达$y$
	+ **弱连通图 Weakly Connected** - 基图连通，本身不连通的有向图
	+ **完全图 Complete** - 任意两个顶点之间都存在边，是连通性最强的图，共有$C_{n}^{2}$条边

+ **有权性 Weightedness**
	+ **带权图 Weighted** - 边带有权值的图
		+ **有符号图 Signed** - 权值有正负号的图
	+ **无权图 Unweighted** - 所有边权值相等的图

+ **子图 Subgraph**
	+ **生成树 Spanning tree** - 可以构成一棵树的最小连通子图，共有$n-1$条边

## 结构实现 Implementation

> **邻接矩阵 Adjacent Matrix**

用二维矩阵结构记录任意两个顶点之间的连接关系。由于其空间复杂度是$O(n^2)$，不适合顶点数较多的稀疏图（Sparse Graph）使用。

表示带权图时，参考距离的定义，常用无穷表示边不存在；表示无权图时，可以直接用布尔值表示两个节点之间是否有边相连；

## 迪杰斯特拉算法 Dijkstra‘s Algorithm
