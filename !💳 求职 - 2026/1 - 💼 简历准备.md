### 本科毕业设计

> **系留无人机基于强化学习的避障算法
> Obstacle Avoidance for Roped Drone based on Reinforcement Learning

+ 项目以Sim-to-real的框架进行，避障模型在仿真器MuJoCo中进行训练，基于PPO算法生成位置控制目标点，躲避从随机方向靠近的动态障碍物，并在实机上进行测试。
+ The project was in a sim-to-real scheme. The object avoidance model was trained on the MuJoCo simulator using the PPO RL algorithm and could generate position target for the dodging action. The model was tested on a real drone.

+ 根据四旋翼无人机的运动学模型创建了一个可飞行的无人机MJCF模型，调用Python的SimplePID库编写了PID控制器实现无人机的位置控制。
+ Built MJCF drone model based on the kinematic model of quadropters. Implemented a PID controller to achieve position control of the simulated drone using the Python SimplePID package.

+ 调用Python的Gymnasium库编写了强化学习环境的状态、动作空间和奖励项，使用ROS系统的话题订阅机制实现了训练、控制器、仿真器节点之间的通信。经实机测试，对飞行速度为1.5m/s的障碍物实现了85%以上的避障成功率。
+ Defined the RL state space, action space and reward using the Python Gymnasium package. The communication of the trainer, controller and simulator nodes were implemented using the ROS topic mechanism. When tested on real flights, the model reached a dodging success rate of over 85% for obstacles with a velocity of 1.5 m/s.

> **困难与解决方案**

+ **仿真器选择**：仿真环境和模型原本选用的是Gazebo上PX4提供的Iris无人机模型，后来发现Gazebo的环境在重置、运行过程中都存在很大的延迟，而且界面在训练过程中很不稳定，界面经常崩溃。后来与课题组的师兄讨论，转为使用更轻量的MuJoCo仿真器。Gazebo仿真器追求贴近真实的机器人仿真，因此渲染、碰撞计算占用更多资源；而MuJoCo专门为强化学习训练设计，简化了渲染管线和中层件，可以以更快的速率运行仿真。
+ **动捕数据导入ROS系统**：将动捕数据直接导入ROS系统才能实现无人机实机的位置与速度反馈。最终找到了用VRPN（虚拟现实外围网络）接口传输数据的方法。动捕数据可以经由网线从动捕系统的主机直接传入笔记本的Ubuntu虚拟机，ROS官方提供的vrpn_client_ros模块将其以话题的形式广播，由强化学习模块接收作为观测结果。
+ **强化学习奖励项调节**：一开始，无人机靠近障碍物的每步惩罚数值相对于避障成功的奖励数值设置得过高，出现了Reward Hacking现象，无人机主动撞向障碍物，快速结束当前Episode以减少总奖励损失。后来尝试调低每步惩罚数值，却发现训练出的模型避障成功率大幅下降。最后通过精确微调奖励项的衰减函数才让无人机对1.5m/s直线运动的障碍物避障成功率达到85%
