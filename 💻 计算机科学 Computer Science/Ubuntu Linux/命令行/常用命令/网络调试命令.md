
---
## ifconfig：查看IP地址

网络端口及IP地址查看

```bash
ifconfig
```

---
## nmcli：网络管理器

Linux网络管理器命令行接口（Network Manager Command-Line Interface）

打开并刷新所有网络服务，关闭是`off`：

```bash
nmcli n on
```

---
## ping：检查IP端口连接状态

支持IP地址和网址输入

```
ping 192.168.0.1
ping www.baidu.com
ping www.google.com
```
