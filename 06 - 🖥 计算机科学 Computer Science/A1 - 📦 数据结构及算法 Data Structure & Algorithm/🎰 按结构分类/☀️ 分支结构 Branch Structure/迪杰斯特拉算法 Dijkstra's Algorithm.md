迪杰斯特拉算法是一种图节点的最短路径搜索算法，运行一次即可计算从一个节点出发到其余所有节点的最短距离及最短路径。其基本步骤如下：

1. 选择**出发点**$v_s$
	
2. 设置其余节点到出发点的距离为无穷大$\infty$
	
3. 循环访问节点并更新最短距离，直到所有节点都已访问过
	
	+ 3.1 访问当前已知距离与出发点最近的**未访问节点**，记为**当前节点**$v_c$
		
	+ 3.2 **更新最短距离与路径**：若某节点$v$可由**当前节点**直接到达，且该路径**路程更短**，更新其到出发点的最短距离为出发点到当前节点距离与当前节点到该节点距离之和。到节点$v$路径的前置节点更新为当前节点
		
	+ 3.3 当前节点标记为已访问节点

迪杰斯特拉算法仅适用于**边权重值非负**的图，若图上存在负值的边权重值，则需要使用Bellman-Ford算法
