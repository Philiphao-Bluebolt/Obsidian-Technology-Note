Linux系统下的主要快捷键收录

---
### 文件目录

+ `CTRL` + `H` · 切换显示隐藏文件
+ `CTRL` + `ALT` + `T` · 新命令行窗口（Home目录）
+ `CTRL` + `F10` + `E` · 新命令行窗口（当前目录）

### 命令行

+ `CTRL` + `SHIFT` + `T` · 新命令行子窗口
+ `CTRL` + `SHIFT` + `C` · 复制
+ `CTRL` + `SHIFT` + `V` · 粘贴
+ `CTRL` + `C` · 结束进程
+ `CTRL` + `Z` · 暂停进程（`fg`/`bg` 继续执行）

### 截屏

+ `PRT SC` · 截屏（全屏）
+ `ALT` + `PRT SC` · 截屏（单窗口）
+ `SHIFT` + `PRT SC` · 截屏（框选区域）
+ `CTRL` + `ALT` + `SHIFT` + `R` · 录屏
