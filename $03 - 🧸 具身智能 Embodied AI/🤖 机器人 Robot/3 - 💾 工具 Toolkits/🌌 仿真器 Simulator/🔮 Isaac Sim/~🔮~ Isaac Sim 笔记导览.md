## 安装配置

Isaac Sim同时支持Windows和Ubuntu，但对操作系统版本和硬件性能有一定要求。可以用官方提供的[兼容性检测器]([Isaac Sim Compatibility Checker](https://docs.isaacsim.omniverse.nvidia.com/4.5.0/installation/requirements.html#isaac-sim-compatibility-checker))进行测试，测试通过即可以安装。测试之前，需要先安装两个驱动工具：

+ **NVIDIA 驱动** - Settings --> About --> Software Updates --> Additional Drivers
+ **NVIDIA 显卡检测** - `sudo apt install nvidia-msi-xxx`

通过后就可以安装Isaac Sim了
