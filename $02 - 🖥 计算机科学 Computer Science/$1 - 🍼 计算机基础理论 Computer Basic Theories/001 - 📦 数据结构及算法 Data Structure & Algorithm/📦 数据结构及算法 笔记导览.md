**数据结构**分为**抽象数据结构**（ADT）与**具体数据结构**（CDT）——ADT是数学上规定了储存结构与操作的黑箱模型；CDT则是ADT在内存层面的实现，属于白箱模型。CDT常被封装成可以复用的**容器**（Container），容器内部使用的抽象数据结构可能与容器代表的数据结构本身不同，比如栈的容器可能是用数组实现的。

**操作算法**是对数据结构元素的处理方法，分为**读**、**写**两大类——读操作不改变元素，如遍历、搜索、计数、寻路；写操作则会改变元素顺序或具体数据，如插入、删除、排序、清空。

+ **插入 Insert** - (序列、树、图) 指定位置插入元素；(栈和队列) 压入元素；(映射) 新增键值对
+ **删除 Delete** - (序列、树、图) 删除指定元素或键；(栈和队列) 弹出元素；(映射) 删除指定键值对
+ **清空 Clear** - 删除所有元素
+ **复制 Copy** - 将所有元素复制到另一个容器
+ **遍历 Traverse** - 按一定顺序穷举所有元素，即线性化，本质上和搜索算法相同
+ **搜索 Search** - 寻找指定元素
+ **计数 Count** - 计算指定元素的出现次数
+ **寻路 Path** - 寻找两个元素之间的路径

实际执行时，部分操作需要先执行其他操作才能执行，如二分查找要求先对数组进行排序。

## 顺序搜索 Sequential Search

+ **复杂度**：时间 $O(n)$ - 空间 $O(1)$

顺序搜索是针对线性容器最简单的搜索方法，即从第一个元素搜索到最后一个元素，最坏情况下的时间复杂度为线性复杂度$O(n)$

### 实现代码

> **C++**

```C++
int seqSearch(vector<int>& nums, int target) {
	for (int i = 0; i < nums.size(); i++) {
		if (nums[i] == target) return i;
	}
	return -1;
}
```

## 二分搜索 Binary Search

+ **参考**：[Geeksforgeeks](https://www.geeksforgeeks.org/binary-search/)
+ **复杂度**：时间 $O(\log n)$ - 空间 $O(1)$

二分搜索又称为折半搜索，其基本思想是：在一个**已排序**的序列中，每次只需检测正中间的元素便可将查找范围缩小一半，从而将时间复杂度降低到对数级别，是线性结构查找算法中效率最高的算法。

每一轮迭代的中位元素序号`mid`可以由子序列的最左侧元素序号`low`和最右侧元素序号`high`取平均得到。子序列长度为奇数时`mid`指向中间元素，为偶数时指向中间靠左元素（整型去尾）。

```C++
mid = (low + high) / 2

// oddArr = [1, 2, 3], mid = 2
// evenArr = [1, 2, 3, 4], mid = 2
```

继续迭代的条件需要取等号，否则在两边的指针重叠时会忽略指向值

```C++
while (low <= high)
```

### 实现代码

> **C++**

```C++
int binarySearch(vector<int>& nums, int target) {

    int left = 0, right = nums.size() - 1, mid = 0;
    while (left <= right) {
        mid = (left + right) / 2;
        if (target < nums[mid]) right = mid - 1;
        else if (target > nums[mid]) left = mid + 1;
        else return mid;
    }
	return -1;
}
```

> **Java**

```java
int binarySearch(int arr[], int x) 
    {
        int low = 0, high = arr.length - 1;
        while (low <= high) {
            int mid = low + (high - low) / 2;

            if (arr[mid] == x) // Check if x is at mid
                return mid;

            if (arr[mid] < x) // If x greater, ignore left half
                low = mid + 1;

            else // If x is smaller, ignore right half
                high = mid - 1;
        }

        // If we reach here, then element was not present
        return -1;
    }
```

+ **参考**：[Geeksforgeeks](https://www.geeksforgeeks.org/two-pointers-technique/)

双指针搜索是一种在已排序的线性容器中搜寻元素对的方法，可以将搜索两个元素的时间复杂度降低到线性级别$O(n)$，其基本思想是在遍历过程中跳过的元素对。
