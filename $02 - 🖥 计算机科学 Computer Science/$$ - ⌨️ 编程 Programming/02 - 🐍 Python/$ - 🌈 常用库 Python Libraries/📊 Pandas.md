## DataFrame

DataFrame储存2D表格数据，是Pandas中的原生数据容器，默认以行代表数据样本，列代表样本特征。

### 创建

DataFrame可以通过导入外部数据库或调用构造函数手动创建，也可以由列表、Numpy Array等其他数据容器转换而来。

> **数据导入**

```python
pd.read_excel("data.xlsx")
pd.read_csv("data.csv")
```
