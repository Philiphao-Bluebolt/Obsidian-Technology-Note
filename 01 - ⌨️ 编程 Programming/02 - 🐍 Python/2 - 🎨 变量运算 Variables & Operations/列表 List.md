## 切分 Slicing

列表可以用方括号运算符`[]`与顿号`:`指定序号范围**选取**一部分元素，称为切分。切分需要指定起始编号、终止编号、步长，且默认**左闭右开**，不包含终止编号，与`range()`等函数的设计风格相同。

```python
# [起始序号 : 终止编号 ：步长] 
arr[2:8:2] # 选取元素编号 --> 2 4 6 
```

+ **省略步长**：可仅指定起始、终止编号而省略第二个顿号及步长，此时默认连续选取（步长1）

```python
# [起始序号 : 终止编号]
arr[2:8] # 选取元素编号 --> 2 3 4 5 6 7
```

+ **默认起始**：起始序号留空时默认为从第一个元素开始

```python
# [ : 终止编号]
arr[:8] # 选取元素编号 --> 0 1 2 3 4 5 6 7
```

+ **默认终止**：终止序号留空时默认选取到最后一个元素

```python
# [起始序号 :]
arr[:8] # 选取元素编号 --> 0 1 2 3 4 5 6 7
```

+ **负数编号**：负数的起始与终止序号表示倒数第几个元素，负数步长表示从尾到头倒着数

```python
# 设数组长度为10
arr[-1:-3:-1] # 选取元素编号 --> 9 8
```

下面是一些切分的例子

```python
# arr = [0, 1, 2, 3, 4, 5, 6]
print(arr[1:3]) # [1, 2]
print(arr[:]) # [0, 1, 2, 3, 4, 5, 6]
print(arr[-1:1:-2]) # [6, 4, 2]
arr[2:4] = [3, 2, 2] # --> arr = [0, 1, 3, 2, 2, 4, 5, 6]
```
