给定大小为`2*n`的地图，初始时智能体位于左上角且状态为0，每个回合，智能体可以切换自己的0/1状态，或者往上下、右边状态相同的格子移动一格。问最少几个回合可以移动到地图的右下角？

输入格式如下，第一行为测试地图实例数，每一个测试实例的第一行表示地图宽度，后面两行表示地图结构。

```
2
4
0 1 1 0
0 0 0 0
5
0 1 1 0
1 1 0 1
```

输出格式，每一个实例的最少回合数占一行

```
4
7
```

这是一个简单的找最优路径的问题，由于地图只有两行，猜想可以由局部最优得到全局最优。除了最右侧格子不能继续向右以外，所有格子都有四个行动策略：

+ 向右
+ 竖直移动（向上 / 向下）
+ 改变状态再向上
+ 改变状态再向右
