求给定整数区间$[l,h]$之间因数和为奇的数的个数，举例如下：

>数字10的因数包括1、2、5、10，和为18，不是奇数
>数字8的因数包括1、2、4、8，和为15，是奇数

+ 输入格式：第一行表示测试用例数，后面每一行的两个数分别表示测试区间的下界$l$和上界$h$，且保证$l<h$

```
5
4 20
19 567
646 22222
22 100
1 100000
```

+ 输出格式：

```
1 1 1 1 1
```

---
## 思路

这一题本质上是数论题，使用数学归纳法，列出1～20的因数和就会发现，只有三类数满足条件

+ 完全平方数
+ 二的幂数
+ 完全平方数与二的幂数的乘积

奇偶数的加乘规律：

+ 奇 + 奇 = 偶
+ 奇 + 偶 = 奇
+ 偶 + 偶 = 偶
+ 奇 x 奇 = 奇
+ 奇 x 偶 = 偶
+ 偶 x 偶 = 偶

---
## 代码

+ **复杂度** - $O(\sqrt{n}\log n)$ / $O(\sqrt{n}\log n)$

这一题有两种实现方式，效率较低的实现方式是检测区间内的每一个数，效率较高的实现方式是只考虑平方数、二的幂和两者之积，时间复杂度为$O(\sqrt{n}\log n)$（$n=h-l$）

```cpp
#include <fstream>
#include <iostream>
#include <cmath>
#include <unordered_set>
using namespace std;

int main(){
    freopen("./input.txt", "r", stdin);

    int n_test;
    int low, high;
    int power;
    unordered_set<int> appeared;
    cin >> n_test;
    

    for (int i_test = 0; i_test < n_test; i_test++){
        cin >> low >> high;
        // cout << low << " " << high << "\n";
        appeared.clear();
        for (int i = 1, isq = 1; isq <= high; i++, isq = i*i){
            if (isq >= low && isq <= high && appeared.find(isq) == appeared.end()) appeared.insert(isq);

            power = 2;
            while(power * isq <= high){
                if (isq * power >= low && isq * power <= high && appeared.find(isq*power) == appeared.end()){
                    // cout << isq * power << " ";
                    appeared.insert(isq*power);
                }
                power *= 2;
            }
        }

        // for (int i : appeared) cout << i << " ";
        cout << appeared.size() << "\n";
    }
}
```
