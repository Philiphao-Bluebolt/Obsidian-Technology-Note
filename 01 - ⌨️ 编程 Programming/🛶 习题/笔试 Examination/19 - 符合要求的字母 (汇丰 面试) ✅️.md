给定一串字符串，要求输出符合以下要求的字母个数（大小写算同一个字母）

+ 必须同时出现大写与小写
+ 小写只能出现在大写之前

---
## 思路

这题看似简单，实际写的时候计数逻辑容易写错。下面是实现逻辑

+ 如果遇到小写
	+ 没有出现过大写
		+ 没有出现过小写 -> 记录
		+ 出现过小写 -> Do nothing
	+ 出现过大写
		+ 前面已违规 -> Do nothing
		+ 违规 -> 计数-1
+ 如果遇到大写
	+ 出现过小写 -> 记录，计数+1
	+ 没有出现过小写 -> 记录

---
## Python

+ **复杂度** - $O(n)/O(n)$

```python
import sys
import os

class CountingLetters:
    def __init__(self, strings: list):
        self.strings = strings

    def counting_letters(self, string : str) -> int:
        cnt = 0
        record = set()
        for letter in string:
            # Lower
            if 97 <= ord(letter) <= 122:
                if letter.upper() not in record and letter not in record:
                    record.add(letter)
                elif letter.upper() in record and letter in record:
                    record.discard(letter)
                    cnt -= 1

            # Upper
            elif 65 <= ord(letter) <= 90:  
                if letter.lower() in record and letter not in record:
                    record.add(letter)
                    cnt += 1
        return cnt

    def counting_letters_all(self):
        return tuple(map(self.counting_letters, self.strings))

def main():
    test_strs = []
    os.chdir(os.path.dirname(__file__))
    with open("strings.txt", "r") as file:
        sys.stdin = file
        for line in sys.stdin:
            test_strs.append(line[0:len(line)-1])
    
    print(test_strs)
    cl = CountingLetters(test_strs)
    print(cl.counting_letters_all())
    
if __name__ == "__main__":
    main()
```
