## 知识点

+ ✨️ **算法范式 (Algorithm Paradigms)**
	
	+ **分治法 (Divide and Conquer)**：把较复杂的问题分解为几个较简单的问题，递归是最常见的分治算法
	
+ 🎬️ **通用算法 (Common Algorithms)**
	
	+ **递归 (Recursion)**：函数重复调用自身直至达到终止条件，适合用于解决具有分形逻辑结构的问题，如二分法、二叉树
	
	+ **哈希 (Hashing)**：本质上是以空间换时间，用哈希表记录已经遍历过的数据从而免去重新搜索所需的时间
	
	+ **动态规划 (Dynamic Programming)**：
	
+ 🪈 **数组 (Array)**
	
	+ **双指针 (Two Pointers)**：在已排序数组中，利用首尾指针逼近的方法搜索符合要求的元素对，可以减少无效比较，降低时间复杂度
	
	+ **前缀和 (Prefix Sum)**：遍历过程中逐次记录下前`i`个元素的总和`s[i]`，以便快速计算数组任意分段的总和`ss = s[a] - s[b]`
	
	+ **滑窗 (Sliding Window)**：
	
+ ⛓️‍💥 **链表 (Linked List)**	
	
	+ **假节点 (Dummy Node)**：新链表的`head`不确定时，用一个假节点充当临时`head`，从而避免首节点选取的模糊
	
	+ **龟兔算法 (Hare and Tortoise Algorithm)**：又名弗洛伊德寻环算法，快慢指针同时出发，快指针`fast`一次跳两步，慢指针`slow`一次跳一步，可用于检测链表环路或链表中间节点
