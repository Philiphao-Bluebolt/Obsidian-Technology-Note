Vim是一个基于纯字符输入的命令行文本编辑器，在熟悉其命令的前提下可以达到极高的编辑效率。Vim通过通过复杂的字符命令系统实现其众多功能，简单的字符命令之间可以相互组合

+ [基本使用](#基本使用)


---
## 基本使用

> **普通模式** - 可以输入命令，移动光标，删减文本， 按`:`进入命令模式，按`i`进入插入模式  
> **插入模式** - 可以像一般的文本编辑器一样插入字符编辑，按“Esc”退出
> **命令模式** - 可以输入一些控制命令

### 保存退出

+ `:w` - 保存修改
+ `:q` - 退出
+ `:wq` - 保存并退出
+ `:q!` - 不保存退出

### 编辑文本

+ `x` - 删除一个字符
+ `dd` - 删除整行
+ `yy` - 复制整行
+ `p` - 粘贴
+ `u` - 撤回
+ "Ctrl" + `r` - 重做

### 光标移动

+ `h` / `j` / `k` / `l` - 左下上右（方向键 亦可）
+ `0` / `$` - 行首行末 （"Home" / "End" 键 亦可）	
+ `w` / `b` - 下一个、上一个词
+ `gg`/ `G` - 文件第一行、最末行
+ `:行数` - 第几行行首
+ `f字符` - 移动到**同一行**下一个指定字符
+ `t字符` - 移动到下一个指定字符

### 搜索

+ `/字符串` - 向后搜索
+ `?字符串` - 向前搜索

### 分屏

+ `:split` - 水平（上下）分屏
+ `:vsplit` - 垂直（左右）分屏
+ "Ctrl" + `w` + `h` / `j` / `k` / `l` - 切换分屏（左下上右）（方向键 亦可）
+ `:q` 退出上一个分屏
