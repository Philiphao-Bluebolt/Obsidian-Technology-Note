递归是一种特殊的函数调用方法：函数体在内部调用自身，不断重复一个运算过程，直到达到某一个终止条件才逐层返回，结束递归。所有编程语言都能实现递归。

数学上，递归与循环是等价的逻辑过程，任何以递归形式实现的算法总有对应的循环写法，反之亦然。由于会产生大量的中间变量，递归算法占用的内存空间往往比循环更大，但某些算法的递归形式比循环形式更直观。

## C++递归

```cpp
int resursive(int x, int target){
	x++;
	if (x < target) return resursive(x, target);
	else return x;
}
```
